# Security Policy

## Reporting a Vulnerability

Tom n Jerry is a development tool skill pack. It does not handle secrets or user data directly.

If you discover a security concern — supply-chain risk in the npm package, unsafe file operations in the CLI, or other issues — please open an issue at:

https://github.com/hrshx3o5o6/Tom-n-Jerry/issues

Do not open public issues for critical vulnerabilities. Email the maintainer directly via the email address in the npm package metadata.
